## Details

### 0.1.x — First public release

Native HTTP action semantics, Twig integration, basic validation, Flex recipe

### 0.2.x — Validation and ergonomics

Improved validation messages, additional semantic rules, configuration polish, migration guides

### 0.3.x — Browser semantics

Form ownership, target handling, download handling, rel validation, method semantics, action inheritance

### 0.4.x — Accessibility

Keyboard semantics, focus handling, disabled state validation, ARIA integration, assistive technology guidance

### 0.5.x — Package integration

UX Blocks integration, UI Kernel integration

### 0.6.x — Extensibility

Custom action types, extension points, validator hooks, reusable rule sets, provider contracts

### 0.7.x — Diagnostics

Action inspector, debug commands, semantic reports, project audit, automatic fix suggestions

### 0.8.x — Developer experience

Interactive CLI, improved exceptions, IDE metadata, documentation examples, cookbook, testing helpers

### 0.9.x — API freeze

Deprecation cleanup, documentation review, extension guide, compatibility audit, final migration guide

### 1.0.x — Stable API

Semver-stable action contracts, stable extension API, stable validation policy, complete handbook
