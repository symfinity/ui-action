# Installation

## Requirements

PHP 8.2+. Symfony 7.4 when integrating in a Symfony app.

## Composer

```bash
composer require symfinity/ui-action
```

## Autoloading

```php
require_once dirname(__DIR__).'/vendor/autoload.php';
```

## Verify

Minimal script or PHPUnit smoke.
